# Star Wars API (FastAPI + GCP Cloud Run)

API simples em **FastAPI** que consome a **SWAPI** (Star Wars API) e permite buscar informações de **personagens (people)**, **planetas (planets)** e **naves (starships)** usando filtro (`search`).
